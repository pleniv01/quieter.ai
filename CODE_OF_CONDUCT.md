# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and considerate
- Assuming good faith
- Accepting constructive feedback

Harassment, abuse, or exclusionary behavior is not tolerated.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards.
